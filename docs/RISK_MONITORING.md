# External services

## Vercel
### Risk level: High
### Fallback: none
We use Vercel to host our websites. If this goes down, our websites will not be accessible anymore.

## Supabase
### Risk level: High
### Fallback: skip
We use Supabase to store user data, game data and our store. If this goes down, we will not be able to store user data, game data and people will not be able to buy anything in our game.

## Youform
### Risk level: Low
### Fallback: skip
We use Youform to collect user feedback via forms. If this goes down, we will not be able to collect user feedback.

## GitHub
### Risk level: High
### Fallback: none
We use GitHub to host our code. If this goes down, we will not be able to update source code.

## Spaceship
### Risk level: Medium
### Fallback: none
We use Spaceship as our domain registrar. If this goes down, we will not be able to update our domain.

## Instatus
### Risk level: Medium
### Fallback: none
We use Instatus to display our status page. If this goes down, we will not be able to display our status page.

## Google Play
### Risk level: High
### Fallback: none
We use Google Play to distribute our app. If this goes down, we will not be able to distribute our android app.

## PostHog
### Risk level: Medium
### Fallback: skip
We use PostHog to collect analytics. If this goes down, we will not be able to collect analytics.

## Have I been pwned
### Risk level: Low
### Fallback: skip
We use Have I been pwned to check if a password has been compromised before signup. If this goes down, we will not be able to check if a password has been compromised.
