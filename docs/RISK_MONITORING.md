# External services

## Vercel
### Risk level: High
We use Vercel to host our promotional website and our online play website. If this goes down, we will not be able to promote our game or allow people to play it online.

## Supabase
### Risk level: High
We use Supabase to store user data, game data and our store. If this goes down, we will not be able to store user data, game data and people will not be able to buy anything in our game.

## Formbricks
### Risk level: Low
We use Formbricks to collect user feedback. If this goes down, we will not be able to collect user feedback.

## GitHub
### Risk level: High
We use GitHub to host our code. If this goes down, we will not be able to update source code.

## Spaceship
### Risk level: Medium
We use Spaceship as our domain registrar. If this goes down, we will not be able to update our domain.

## Instatus
### Risk level: Medium
We use Instatus to display our status page. If this goes down, we will not be able to display our status page.

## Google Play
### Risk level: High
We use Google Play to distribute our app. If this goes down, we will not be able to distribute our app.

## PostHog
### Risk level: Medium
We use PostHog to collect analytics. If this goes down, we will not be able to collect analytics.
